# 🚀 Advanced Task Manager App

A lightweight, robust command-line application built with Python to efficiently create, track, and manage your daily tasks. Built with data persistence using JSON storage.

---

## ✨ Features

*   💾 **Persistent Storage**: Automated JSON database backups save your tasks securely.
*   ⚡ **High Performance**: Highly optimized core routines for instant data lookup.
*   📊 **Clean Architecture**: Designed with decoupled functions for seamless scalability.
*   👥 **Co-Authored Core**: Developed in collaboration to ensure code compliance.

---

## 🛠️ Tech Stack

*   **Language**: Python 3.x
*   **Database**: JSON File System
*   **Version Control**: Git & GitHub

---

## 📦 Installation & Setup

1. **Clone the repository to your machine**:
   ```bash
   git clone https://github.com
   ```

2. **Navigate into the project directory**:
   ```bash
   cd task-manager-app
   ```

3. **Run the core application directly**:
   ```bash
   python app.py
   ```
